# MathTrainer 🎯  
An iOS app for practicing math skills  

## 📱 Overview  
MathTrainer is a learning project designed to help users practice basic arithmetic operations: **addition, subtraction, multiplication, and division**.  
The user selects an operation, solves random math problems, and tracks their progress. Results are saved both locally and in **Firebase Firestore**.  

## 🛠️ Technologies  
- Swift  
- UIKit  
- Storyboard  
- AutoLayout  
- Firebase Firestore  

## 📚 Fundamental Concepts  
This project demonstrates:  
- Working with `UIViewController` and `UIView`  
- Using **Segues** for navigation between screens  
- Understanding **intrinsic content size**  
- Applying **AutoLayout constraints and priorities**  
- Managing data with **UserDefaults**  
- Integrating **Firebase Firestore** for cloud storage

## 🚀 Learning Goals  
This project was built to practice and demonstrate:  
- Setting up a **Git repository**  
- Building an iOS app from scratch  
- Designing interfaces with **Interface Builder**  
- Connecting UI elements via `IBOutlet` and `IBAction`  
- Handling screen transitions (**Segue, unwind**)  
- Saving and syncing data locally and remotely  
- Exploring **AutoLayout** and **intrinsic content size** in practice

## ☁️ Firebase Setup  
⚠️ For security reasons, the file `GoogleService-Info.plist` is **not included** in this repository.  
To run the app:  
1. Create a Firebase project in [Firebase Console](https://console.firebase.google.com).  
2. Download your own `GoogleService-Info.plist`.  
3. Place it in the project root (`MathTrainer/MathTrainer/`).
